# nmap-network-scan-
Basic network scanning project using Nmap to identify open ports and services with analysis and results.

# 🔍 Basic Network Scanning with Nmap

## 📌 Objective

To perform a network scan using Nmap and identify open ports and services running on a system.

## 🛠️ Tools Used

* Nmap

## ⚙️ Steps Performed

1. Installed Nmap on the system
2. Performed a basic scan on the local machine
3. Used service detection (`-sV`) to identify services
4. Saved output in a text file

## 📊 Scan Results

The following open ports were identified:

* **Port 22 (SSH)** – Used for secure remote access
* **Port 80 (HTTP)** – Used for web communication
* **Port 443 (HTTPS)** – Used for secure web communication
